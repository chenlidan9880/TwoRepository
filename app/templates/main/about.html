关于校园网络监控系统

系统概述

校园网络监控系统是一个专为高校网络环境设计的综合性网络管理平台，旨在提供直观、高效的网络基础设施监控和管理解决方案。该系统基于Python Flask框架开发，采用前后端分离架构，专注于解决校园网络环境中设备管理、流量监控、安全预警等核心需求。

主要功能
设备管理

支持对路由器、交换机、服务器等各类网络设备的集中管理和状态监控。

流量监控

实时展示网络设备的入站/出站流量，提供多维度的流量趋势分析。

告警管理

当设备状态异常或流量超过预设阈值时自动生成告警，支持多级别告警分类。

终端管理

展示连接到网络设备的各类终端设备，如PC、移动设备、IoT设备等。

流量热力图

直观展示整个网络中各设备的流量负载情况，快速定位高负载区域。

网络拓扑

可视化展示网络设备之间的连接关系，帮助理解网络结构。

技术栈
Python Flask
SQLAlchemy MySQL
Bootstrap ECharts
